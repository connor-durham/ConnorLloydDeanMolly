Beer Frontier: About Us

Created by a student team from UCF Coding Bootcamp, Beer Frontier is a web application that will elevate your night out to the next level. Essentially, Beer Frontier allows the user to find different types of beer in their area by searching by beer name, brewery name, location, ingredients, events, etc. Frontier is completely mobile compatible due to integrating Materialize CSS into its programming allowing the user to find their beers on the go.
